<p><p>A Mach Zehnder Interferometer (MZI) is an optical device used to measure small changes in refractive index, which can be used in various applications such as telecommunications, sensing, and metrology. It consists of a beam splitter that splits an input beam into two arms, which then recombine at an output photodetector, producing an interference pattern. By analyzing the interference pattern, small changes in the refractive index of one of the arms can be detected.</p><p>The current outlook of the MZI market is positive.
